Spend gate policy

No outbound spend executes without passing the gate. The policy is evaluated on every SpendTicket.

# POLICY-DATA-REFINE-01
if ticket.amount <= job.auto_approve_threshold:
    execute() # logged, no human needed
else:
    require_human_approval()  # pauses at gate
always:
    audit_log.append(action, actor, details)

Audit certificate

Every completed job produces a signed certificate — the deliverable customers pay for. Available as JSON and PDF.

cert_id        string     # cert_7a2b9c4d81
job_id         string     # JOB-2025-00147
issued_at      datetime
ledger         object     # revenue, gated_spend, approvals
decisions[]    array      # action, actor, amount
dataset        object     # rows, quality, pii_masked
signature      string     # ed25519
