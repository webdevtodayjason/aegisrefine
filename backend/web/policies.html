The standing rule set every refinement job is governed under. These clauses are enforced by the live backend.

◭
Local inference is uncapped
Aegis-14B triage, scoring & planning run on DGX Spark with no spend gate and no human stop.
⛉
Every outbound dollar is gated
Any external/paid tool spend pauses for explicit human approval. There is no auto-spend path — currently enforced at a $0.00 threshold.
✍
Approver identity is recorded
Approve/deny carries X-Admin-User; the handle is written to the immutable audit trail on every decision.
⛉
Decisions are signed
Each gate decision & each finished job ships an Ed25519-signed record (AAR), independently verifiable at L2.
⚑
Execution is a test stub
Approved spend "executes" without moving real money. Live payout via @stripe/link-cli lands in the SPEND milestone.
